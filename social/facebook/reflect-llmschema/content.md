# C++ to LLM tool-use schemas, at compile time
In the AI agent era every tool you expose to Claude or GPT needs a JSON-Schema description. One consteval call emits the tool-use JSON: parameter names, types, docstrings, dispatch, done. The library is reflect_llmschema.
