Generated by `/advertise-post <slug>` (see `.claude/skills/advertise-post/SKILL.md`).

Layout:

```
social/
├── linkedin/<slug>/
│   ├── caption.md         # paste-ready post copy + hashtags + alt-text + suggested time
│   ├── image.png          # 2400x2400 branded card (drop into LinkedIn upload)
│   ├── content.md         # source for the brand-gen card
│   ├── config.yaml        # brand-gen project config
│   └── assets/            # vendored CSS + logo (per project)
└── facebook/<slug>/       # same shape
```

## Manual posting workflow

Posting is manual on purpose. The skill stops short of hitting the LinkedIn UGC API or the Facebook Graph API; you copy + upload yourself.

For each platform:

1. Open the platform's "create post" UI.
2. Paste the body of `caption.md` (everything between the `## Body` header and the `## Hashtags` header).
3. Append the hashtags from the `## Hashtags` section as a final line.
4. Upload `image.png`.
5. Fill the alt-text from the `## Alt-text` section.
6. Schedule for the time printed under `## Suggested post time` (or post immediately if you missed the window).

## Caption file anatomy

```
# <Post title>

## Body
<2-3 paragraphs, opening hook, CTA>

## Hashtags
#cpp26 #reflection #...

## Alt-text
<one-sentence visual description>

## Suggested post time
Tuesday 2026-MM-DD, 10:00 CET
Reason: ...
```

## Regenerating

If you tweak the wro.cpp brand kit (`.claude/skills/advertise-post/brand-kit/`) or the post's frontmatter, re-run `/advertise-post <slug>` to regenerate. The skill is idempotent at the project-folder level (it'll overwrite content.md / image.png in place).
